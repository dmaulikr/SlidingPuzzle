# SlidingPuzzle
Sliding Image Puzzle IOS Swift image puzzle game.


“SlidingPuzzle” is a beautiful 4*4 sliding image puzzle IOS App, written in swift. Try to solve puzzles is relaxing, rewarding, and keeping your mind active.
